Introduction
~~~~~~~~~~~~~

The 'serial communication manager' is a cross-platform, scalable, performance efficient, portable, thread based, reliable and light weight library designed and developed to exchange data on serial port. It supports RS-232 control signals monitoring and has been ported to Linux, MAC, Solaris and Windows operating system.
